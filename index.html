Requirements

There are system requirements in order to install and setup Inbio theme and its components properly. Make sure that you are running the latest version of WordPress, PHP version 5.6 or higher and MySQL version 5.6 or higher. We also recommend the following PHP configuration limits. https://wordpress.org/about/requirements/

Recommended PHP configuration limits

                                        
                                            max_execution_time = 300;
                                            upload_max_filesize = 128M;
                                            post_max_size = 128M;
                                            max_input_time = 300;
                                            memory_limit = 256M;

Installing the theme

Important! if you download the All Files archive from ThemeForest and try to upload it to your WordPress, you will receive this error: Theme is missing the style.css stylesheet. Please make sure you unzipped the main archive and are uploading the correct file to your WordPress.

When you are ready to install a theme, you must first upload the theme files and then activate the theme itself. The theme files can be uploaded in two ways:

How to Setup and Demo Import (Video): https://youtu.be/ctsT5Y-InqE

1. Using Theme Uploader

Follow the steps as instructed in the images below:

  1. From the dashboard go to AppearanceThemesAdd New Upload Theme
  2. Click on “Choose File” and select inbio.zip
  3. Click on “Install Now”.
  4. Active the Theme

2. Using FTP Clients

Follow the following steps to upload your theme using FTP clients:

  1. Use an FTP such as FileZilla and go to your WordPress installation director
  2. Navigate to /wp-content/themes/ folder and upload the theme folder there
  3. Log in to your WP Admin Dashboard and open the AppearanceThemes menu
  4. You will now see Inbio Theme listed among the other themes. Click on Activate and you’re done!

3. Updating the theme

We recommend that you use the Envato Market plugin to keep the theme up-to-date.

  1. Download the Envato Market plugin from here: http://envato.github.io/wp-envato-market/dist/envato-market.zip
  2. Install the plugin from PluginsAdd New
  3. Once installed, you should see an Envato Market menu on the admin sidebar. Click on it.
  4. Enter your Envato API Personal Token in the Token field. If you don’t have an API token you can create one here or click on the generate a personal token link on that page.
  5. Once you have pasted your token in the Token field, click on Save Changes.
  6. Should your token be correct, two extra tabs are now available on your Envato Market page. Click on the Plugins tab.

If an update is available for the Inbio Theme, click on the Update Available link, which should start installing your update.

Envato Market Plugin Official Site: https://envato.com/market-plugin/

4. Installing required plugins

Inbio theme requires a set of core and additional plugins to function correctly. A plugin is an additional component that offers functionality and features beyond a typical WordPress installation. Without them, most of its functionality is not available. After installing the theme you will be prompted to install the required plugins at the top of your dashboard.

  1. From the dashboard go to Appearance Install Plugins
  2. Toggle all the required plugins and hint “Install”;
  3. The last step will be activating the plugins.

One page Navigation

  1. Prepare the page
  2. Create a new page. To do that go to Pages Add New and give the page a title and then add a Block element to the content. Add section unique "ID" into Per Block.
  3. Prepare the menu
  4. Go to Appearance Menu click the create a new menu link Give the new menu a unique name and then click the Create Menu button.
  5. Add menu item with section ID
  6. Select Onepage menu for this page from page attributes options

For different Home Pages, there is the pre-made menu.

  • home - For Main Demo Page
  • Main Menu - For Inner Page/ Multi-Page Demo
  • Consulting - For Sidenav menu
  • - and more

You can also use external link with one page menu.

  1. Go to Appearance Menu click the menu item arrow icon
  2. Select Yes form External link

You can use SVG Icon / Image For Menu Items

  1. Go to Appearance Menu click the menu item arrow icon Upload a SVG or a Image for the menu.
